This page shows the IPv6 capability status for the top 500 websites according to Alexa.

The data is gathered by first screen-scraping the 500 web sites from Alexa. Unfortunately, I cannot share the free URL provided by Alexa, but as you can see below, the pricing throug their paywall is much more reasonable than previously noted.

http://www.alexa.com/topsites/global

The site names are then checked for AAAA records via the raw name and also www.. For sites which have an AAAA record, a further test is done to see if a web server answers at that name using LWP from an IPv6 source address.

Each column is tallied and totals (and percentages) are shown at the bottom of the table.

We will soon be using the Alexa list again. Amazon has been kind enough to provide free access for this purpose, but it's in a different format which will require me to update the script.

It also turns out that while the pricing I commented on was an accurate reflection of what was on the Amazon web pages at the time, it was not an accurate reflection of the actual pricing. The Alexa 500 subscription price is only $19.99/month and API access can be obtained for $0.0025 per list entry (as opposed to $0.25 as previously reported). Amazon has since updated their web site as well.
